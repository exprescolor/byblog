#### 1.有一对兔子，从出生后第三个月起每个月都生一对兔子，小兔子长到第三个月后每个月又生一对兔子，假如兔子都不死，问每个月的兔子总数为多少？

```java
//这是斐波那契数列，规律是从第三个数开始，每个数都是前两个数的和，最简单的方法就是递归
import java.util.Scanner;

public class Test2{
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		System.out.println("请输入月份");
		int month = sc.nextInt();
		for(int i=0;i<=month;i++) {
			System.out.println(i+"月份的兔子有："+func(i));
		}
	}

	public static int func(int month) {
		if (month==1 || month==2) {
			return 1;
		}else {
			return func(month-1)+func(month-2);
		}
	}
}
```

#### 2.判断101~200之间有多少个素数，并输出所有素数。

```java
//判断素数的方法：用一个数分别去除2和sqrt（这个数本身），如果能被整除，则表明这个数不是素数，反之则是素数
public class Test2{
	public static void main(String[] args) {
		for(int i=101;i<=200;i++) {
			if (idPrime(i)) {
				System.out.println(i);
			}
		}
	}

	public static boolean idPrime(int i) {
		for(int j=2;j<=Math.sqrt(i);j++) {
			if (i%j==0)
				return false;
		}
		return true;
	}
}
```

#### 3.打印出所有的水仙花数，所谓水仙花数是指一个三位数，其各位数字的立方和等于该数本身。

```java
import java.util.Scanner;

public class Test2{
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		System.out.println("请输入一个三位数：");
		int num = sc.nextInt();
		if (isNarcissisticNum(num)) {
			System.out.println("是一个水仙花数");
		}else {
			System.out.println("不是一个水仙花数");
		}
	}
	
	public static boolean isNarcissisticNum(int num) {
		int a=num/100;//取百位
		int b=num/10%10;//取十位
		int c=num%10;//取个位
		if (a*a*a+b*b*b+c*c*c==num) {
			return true;
		}
		return false;
	}
```

#### 将一个正整数分解质因数，例如：输入90，打印出`90=2*3*3*5`

```java
//对n进行分解质因数，应该先找到一个最小的质数i
//如果这个质数=n，则说明分解质因数的过程以已经完成，打印即可
//如果n>i，但n能被i整除，则应打印出i的值，并用n除以i的商，作为新的正整数n，重复执行第一步
//如果n不能被整除，则用i+1作为i的值，重复上一步
import java.util.Scanner;

public class Test2{
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		System.out.println("请输入n的值：");
		int n = sc.nextInt();
		System.out.print("分解质因数："+n+"=");
		fenjie(n);
	}

	public static void fenjie(int n) {
		for(int i=2;i<=n;i++) {
			if (n%i==0) {
				System.out.print(i);
				if (n!=i) {
					System.out.print("*");
				}
				fenjie(n/i);
			}
		}
		System.exit(0);
		
	}
}
```

#### 利用条件运算符的嵌套完成如下要求：学生成绩>=90的同学用A表示，60~89之间的用B表示，60以下的用C表示

```java
import java.util.Scanner;

public class Test2{
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		System.out.println("请输入学生成绩：");
		int score = sc.nextInt();
		if (score>=90) {
			System.out.println("A");
		}else if (score>=60 && score<90) {
			System.out.println("B");
		}else {
			System.out.println("C");
		}
	}
}
```
