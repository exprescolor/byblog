# Java基础

## 流

### 输入/输出流的分类

- java.io包中定义了多个流类型（类或抽象类）来实现输入/输出功能；可以从不同角度对其进行分类
   - 按数据流的方向不同可分为输入流和输出流
   - 按照处理数据单位不同可分为字节流和字符流
   - 按照功能不同可分为节点流和处理流

- JDK所提供的所有流类型都位于包java.io内，都分别继承自以下四种抽象流类型

|字节流|字符流
:-:|:--:|--:
输入流|InputStream|Reader
输出流|OutputStream|Writer
