# Github简明教程

#### 1.git clone


git clone /path/to/repository //创建一个代码仓库的克隆


#### 2.git add


git add filename 
git add *  //添加文件到暂存区


#### 3.git commit


git commit -m 'messages' //提交代码变动


#### 4.git push


git push //将代码改动提交到远端仓库


#### 5.git remote add orign


git remote add orign server //添加远端server
