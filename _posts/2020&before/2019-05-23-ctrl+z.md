# ctrl+z 命令

Ctrl + z 命令是去暂停当前的进程，将其“悬挂”起来，也就是放到waitting list 里面

* `Ctrl + z` to interrupt process

* ```shell
  job   # show interrupted process
  ```

* ```shell
  bg %N  # where N denotes pid , make process continue in background
  ```

* ```shell
  fg %N  # make process work forword
  ```
