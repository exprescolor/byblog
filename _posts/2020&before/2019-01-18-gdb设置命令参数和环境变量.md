# gdb设置命令参数和环境变量

命令参数

```
(gdb) set args XXXX
```

环境变量：
```
(gdb) set environment LD_LIBRARY_PATH=. 
```


如果是在执行的时候设置环境环境变量：
```
export LD_LIBRARY_PATH=.
```
输出这句之后，在去运行程序
