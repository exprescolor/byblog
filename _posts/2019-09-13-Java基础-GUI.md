# GUI

### AWT

- AWT(Abstract Window Toolkit)包括了很多类和接口，用于Java Application的GUI（Graphics User Interface 图形用户界面）编程
- GUI的各种元素（如：窗口、按钮、文本框等）由Java类来实现
- 使用AWT所涉及的类一般在java.awt包及其子包中
- Container和Component是AWT中的两个核心类
