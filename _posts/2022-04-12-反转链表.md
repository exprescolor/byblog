# 题目描述

给定一个单链表的头节点head，请你反转这个链表并返回它。

# 解题思路1——递归

* 先把前节点置空，把head赋给当前节点；

* 然后next = prev，prev = curr， curr = next

* 最后返回previous即可。


算法时间复杂度：O(n),其中n是链表长度；空间复杂度：O(1)。


# 代码如下

```CPP
ListNode* reverseList(ListNode* head)
{
    ListNode* prev = nullptr;
    ListNode* curr = head;
    while(curr)
    {
        ListNode* next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}
```
