# 再看链表

## 单向链表

元素不能随机访问: 访问元素最好O(1), 最坏O(N), 平均O(N/2)=O(N)

* 链表结构

```c++
struct ListNode {
	int val; // 节点的值
	struct ListNode *next; // 下一个节点
 };
```

* 链表初始化

```c++
// 定义空链表
ListNode *head = nullptr;
// 定义新的链表
ListNode *head = new ListNode;
// 简洁方式
ListNode *head = new ListNode(0); // 直接把0作为value给第一个节点
```
