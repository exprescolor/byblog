# 各种数组排序C++版

先看结论：

| 排序算法 | 平均时间复杂度 |
| :------: | :------------: |
| 冒泡排序 |     O(N^2)     |
| 选择排序 |     O(N^2)     |
| 插入排序 |     O(N^2)     |
| 希尔排序 |    O(N^1.5)    |
| 快速排序 |    O(NlogN)    |
| 归并排序 |    O(NlogN)    |
|  堆排序  |    O(NlogN)    |
| 基数排序 |   O(d(n+r))    |

### 冒泡排序

```C++
for(int i = 0 ; i < arr.size(); i++){
  for(int j = i+1; j < arr.size(); j++){
    if(arr[j - 1] > arr[j]){
      swap(arr[j], arr[j- 1]);
    }
  }
}
```





### 选择排序

每次遍历，找到最小的数值与第一个元素交换

```c++
for(int i = 0; i < arr.size(); i++){
  int minIndex = i;
  for(int j = i + 1; j < arr.size(); j++){
    if(arr[j] < arr[minIndex]){
      minIndex = j;
    }
  }
  if(minIndex != i){
    swap(arr[i], arr[minIndex]);
  }
}
```

### 快速排序

先寻找一个pivot，然后和pivot比大小，分成两堆。重复该过程直至每堆只有一个数字

```c++
void Qsort(int *arr, int left, int right){
```
