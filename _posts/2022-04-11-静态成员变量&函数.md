## 静态成员变量与普通成员变量的区别

* 普通成员变量在类中被定义后，不同对象的变量值不同；
* 静态成员变量被定义后，不同对象的值都是这个变量值。

* **请注意！** sizeof()运算符不会计算静态成员变量！！
```CPP
class CMyclass
{
    int n;
    static int s;
};
// sizeof(CMyclass) = 4 而**不是8**！
```
