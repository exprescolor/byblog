# Java基础

#### 反射（reflect）

- 反射机制用来描述所有的类。
   - class：描述类本身
   - Field：描述类中的属性
   - Method：描述类中的方法
   - Constructor：描述类中的构造方法
   - Package：，描述类中的包

- 上述五个组成了反射机制
