npm init

初始化json文件

当安装时会自动把依赖项存到package中



npm install把所有的(package.json)依赖项下载出来













 

```shell
npm init -y
快速生成
npm install 把依赖项全部安装
npm install 包名
npm install --save 包名 
//保存依赖项
npm uninstall 
//删除包
npm uninstall  --save
//删除包名和依赖项
npm  help
//查看指定的帮助
npm 包名 help
//查看制定的包名的帮助
```
