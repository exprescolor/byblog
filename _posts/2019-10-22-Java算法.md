#### 6.输入两个正整数m和n，求其最大公约数和最小公倍数

- 分析：利用辗除法。例如：求4453和5767的最大公约数时，可作如下除法.

```
5767÷4453=1余1314
4453÷1314=3余511
1314÷511=2余292
511÷292=1余219
292÷219=1余73
219÷73=3
```

- 于是得知，5767和4453的最大公约数是73。

```java
package redis;

import java.util.Scanner;

public class Test3 {

	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		System.out.println("请输入第一个数字：");
		int nextInt = scanner.nextInt();
		System.out.println("请输入第二个数字：");
		int nextInt2 = scanner.nextInt();
		System.out.println("最大公约数是："+MaxCommonDivisor(nextInt, nextInt2));
		System.out.println("最小公倍数是："+MinCommonMultiple(nextInt, nextInt2));
	}
	
	public static int MaxCommonDivisor(int nextInt,int nextInt2) {
		if (nextInt<nextInt2) {
			int temp=nextInt;
			nextInt=nextInt2;
			nextInt2=temp;
		}
		while(nextInt%nextInt2!=0) {
			int temp=nextInt%nextInt2;
			nextInt=nextInt2;
			nextInt2=temp;
		}
		return nextInt2;
	}
	
	public static int MinCommonMultiple(int nextInt,int nextInt2) {
		return nextInt*nextInt2/MaxCommonDivisor(nextInt,nextInt2);
	}
}
```

#### 7.输入一行字符，分别统计出其中英文字母、空格、数字和其他字符的个数

```java
package redis;

import java.util.Scanner;

public class Test4 {

	public static void main(String[] args) {
		System.out.println("请输入：");
		Scanner sc = new Scanner(System.in);
		String s = sc.nextLine();
		
		//letter 字母
		//symbol 符号
		//space 空格
		//number 数字
		int letter=0,symbol=0,space=0,number=0;
		char[] array = s.toCharArray();
		for(int i=0;i<array.length;i++) {
			char c = array[i];
			if (('a'<=c&&c<='z')||('A'<=c&&c<='Z')){
				letter++;
			}else if (c>=48&&c<=57) {
				number++;//1-10的ASCII码在48-57之间
			}else if (c==' ') {
				space++;
			}else {
				symbol++;
			}
		}
		
		System.out.println("字母出现的次数："+letter);
		System.out.println("数字出现的次数："+number);
		System.out.println("空格出现的次数："+space);
		System.out.println("符号出现的次数："+symbol);
	}
}
```
