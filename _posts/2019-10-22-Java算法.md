#### 6.输入两个正整数m和n，求其最大公约数和最小公倍数

- 分析：利用辗除法。例如：求4453和5767的最大公约数时，可作如下除法.

```
5767÷4453=1余1314
4453÷1314=3余511
1314÷511=2余292
511÷292=1余219
292÷219=1余73
219÷73=3
```

- 于是得知，5767和4453的最大公约数是73。

```java
package redis;

import java.util.Scanner;

public class Test3 {

	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		System.out.println("请输入第一个数字：");
		int nextInt = scanner.nextInt();
		System.out.println("请输入第二个数字：");
		int nextInt2 = scanner.nextInt();
		System.out.println("最大公约数是："+MaxCommonDivisor(nextInt, nextInt2));
		System.out.println("最小公倍数是："+MinCommonMultiple(nextInt, nextInt2));
	}
	
	public static int MaxCommonDivisor(int nextInt,int nextInt2) {
		if (nextInt<nextInt2) {
			int temp=nextInt;
			nextInt=nextInt2;
			nextInt2=temp;
		}
		while(nextInt%nextInt2!=0) {
			int temp=nextInt%nextInt2;
			nextInt=nextInt2;
			nextInt2=temp;
		}
		return nextInt2;
	}
	
	public static int MinCommonMultiple(int nextInt,int nextInt2) {
		return nextInt*nextInt2/MaxCommonDivisor(nextInt,nextInt2);
	}
}
```

#### 7.输入一行字符，分别统计出其中英文字母、空格、数字和其他字符的个数

```java
package redis;

import java.util.Scanner;

public class Test4 {

	public static void main(String[] args) {
		System.out.println("请输入：");
		Scanner sc = new Scanner(System.in);
		String s = sc.nextLine();
		
		//letter 字母
		//symbol 符号
		//space 空格
		//number 数字
		int letter=0,symbol=0,space=0,number=0;
		char[] array = s.toCharArray();
		for(int i=0;i<array.length;i++) {
			char c = array[i];
			if (('a'<=c&&c<='z')||('A'<=c&&c<='Z')){
				letter++;
			}else if (c>=48&&c<=57) {
				number++;//1-10的ASCII码在48-57之间
			}else if (c==' ') {
				space++;
			}else {
				symbol++;
			}
		}
		
		System.out.println("字母出现的次数："+letter);
		System.out.println("数字出现的次数："+number);
		System.out.println("空格出现的次数："+space);
		System.out.println("符号出现的次数："+symbol);
	}
}
```

#### 8.求s = a + aa + aaa + aaaa + aa...a的值，其中a是一个数字。例如2 + 22 + 222 + 2222 + 22222(此时共有5个数相加)，几个数相加有键盘控制。

```java
package redis;

import java.util.Scanner;

public class Test5 {

	 public static void main(String[] args) {
	        Scanner sc=new Scanner(System.in);
	        System.out.println("请输入a的值：");
	        int a=sc.nextInt();
	        System.out.println("请输入n的值：");
	        int n=sc.nextInt();
	        
	        int t=0,num=0;
	        for(int i=1;i<=n;i++){
	            t=t+a;
	            a=a*10;
	            num=num+t;
	        }
	        System.out.println("最后的值为:"+num);
	    }
}
```

#### 9.一个数如果恰好等于它的因子之和，这个数就称为"完数"。例如6=1＋2＋3.编程 找出1000以内的所有完数。 

```java
package redis;

public class Test6 {

	public static void main(String[] args) {
		System.out.println("1000以内的完数有：");
		for(int i=1;i<=1000;i++) {
			int sum=0;
			for(int j=1;j<=i/2;j++) {
				if (i%j==0) {
					sum=sum+j;
				}
			}
			if (sum==i) {
				System.out.println(i);
			}
		}
	}
}
```

#### 10.一球从100米高度自由落下，每次落地后反跳回原高度的一半；再落下，求它在 第10次落地时，共经过多少米？第10次反弹多高？ 

```java
package redis;

public class Test7 {

	public static void main(String[] args) {
        double s=0;
        double h=100;
        for(int i=1;i<=10;i++){
            s=s+h;
            h=h/2;
            s=s+h;
        }
        System.out.println("共经过"+s);
        System.out.println("第10次反弹的高度"+h);
    }
}
```
