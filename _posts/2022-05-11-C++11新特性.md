# 统一的初始化方法——赋值符号"="可省略

Talk is cheap. Show you the code : )

```CPP
int arr[3] {1, 2, 3};   //C++11可以直接这么写，而不需要括号。
vector<int> a {1, 2, 3};    //vector 同理
string str {"Today"};       //stirng 同理
unordered_map<string, int> pairs { {"a", 1}, {"b", 2}};
int *p = new int[20] {1,2,3};
```
