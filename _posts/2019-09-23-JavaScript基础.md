#### JavaScript

- ECMAScript是JavaScript的标准，所以一般情况下这两个词我们认为是一个意思
- 一个完整的JavaScript实现应该有：**ECMAScript、DOM（文件对象模型）、BOM（浏览器对象模型）**三部分组成

#### JS特点

- 解释性语言
- 类似于C和Java的语言结构
- 动态语言
- 基于原型的面向对象
