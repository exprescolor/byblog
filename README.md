## 模板套用

如果打算用它作blog模板的话，也可以的，fork到本地仓库后，如果你拥有属于自己的域名，可以直接修改CNAME文件中制定的blog地址
如果没有的，可以删掉CNAME，然后更名为"你的github姓名.github.io"，就可以访问你的blog啦
